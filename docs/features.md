# Features

## Lead Management

- Add leads
- Edit lead details
- Delete leads
- Track lead source
- Set lead status
- Assign leads to users

## Customer Management

- Convert leads to customers
- Manage customer profiles
- Store contact details
- Track customer history

## Follow-up Management

- Add follow-up reminders
- Set follow-up date
- Mark follow-up as completed
- Filter pending follow-ups

## Sales Pipeline

- Track deal stages
- Manage deal value
- Update deal status
- View pipeline summary

## Notes

- Add notes to leads and customers
- Track communication history

## Dashboard

- Total leads
- Total customers
- Pending follow-ups
- Active deals
- Won deals
- Lost deals

## Future Enhancements

- Email notifications
- Calendar integration
- Lead import from CSV
- Report export
- React frontend
