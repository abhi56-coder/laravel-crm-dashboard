# Laravel CRM Dashboard

A Laravel-based CRM dashboard for managing leads, customers, follow-ups, sales pipeline, notes, and business reports.

## Project Goal

This project is designed to demonstrate Laravel skills through a SaaS-style customer relationship management workflow.

## Tech Stack

- Laravel 10/11
- PHP 8+
- MySQL
- Blade / Bootstrap
- React optional in phase 2
- Git + GitHub

## Planned Features

- User authentication
- Lead management
- Customer management
- Follow-up reminders
- Sales pipeline tracking
- Notes and comments
- Status filters
- Dashboard reports
- Role-based access planning

## Main Modules

1. Authentication
2. Dashboard
3. Leads
4. Customers
5. Follow-ups
6. Sales Pipeline
7. Notes
8. Reports

## Database Tables

- users
- leads
- customers
- follow_ups
- deals
- notes
- activity_logs
